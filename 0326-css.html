CSS盒模型详解

CSS盒模型是网页布局的基础，每个HTML元素都被视为一个矩形盒子。盒模型由四个部分组成：内容(content)、内边距(padding)、边框(border)和外边距(margin)。

1. 设置盒子大小（宽度和高度）

首先，我们设置盒子的基本尺寸，即宽度(width)和高度(height)：

.basic-box {
  width: 200px;
  height: 100px;
  background-color: #4cc9f0;
}
内容区域

这是一个基础的盒子，只设置了宽度和高度。

2. 设置内边距(padding)

内边距是内容区域与边框之间的空间：

.padding-box {
  width: 200px;
  height: 100px;
  background-color: #90be6d;
  padding: 20px; /* 四个方向都添加20px的内边距 */
}
带内边距的盒子

添加内边距后，盒子的实际占用空间变大了。现在盒子的总宽度是 200px + 20px(左内边距) + 20px(右内边距) = 240px。

3. 设置边框(border)

边框围绕在内边距和内容外部：

.border-box {
  width: 200px;
  height: 100px;
  background-color: #f8961e;
  border: 5px solid #f72585; /* 添加5px宽的实线边框 */
}
带边框的盒子

添加边框后，盒子的实际占用空间进一步增加。现在盒子的总宽度是 200px + 5px(左边框) + 5px(右边框) = 210px。

4. 设置外边距(margin)

外边距是盒子与其他元素之间的空间：

.complete-box {
  width: 200px;
  height: 100px;
  background-color: #4361ee;
  padding: 20px;
  border: 5px solid #f72585;
  margin: 20px auto; /* 上下20px，左右自动居中 */
}
完整盒模型

完整的盒模型包括内容、内边距、边框和外边距。现在盒子的总宽度是 200px(内容) + 40px(内边距) + 10px(边框) = 250px，再加上左右的外边距。

5. 使用box-sizing改变盒模型计算方式

默认情况下，width和height只包括内容区域(content-box)。使用box-sizing: border-box可以改变这一行为：

.border-box-sizing {
  box-sizing: border-box; /* 改变盒模型计算方式 */
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #f72585;
}
border-box盒模型

使用border-box后，width和height包含了padding和border。所以即使添加了内边距和边框，盒子的总宽度仍然是200px。

CSS信息流与显示方式

CSS中的每个元素都有一个默认的display属性值，它决定了元素如何在页面上显示。通过修改display属性，我们可以改变元素的显示方式。

1. display: block（块级元素）

块级元素会占据一整行，可以设置宽度和高度。

.display-block {
  display: block;
  width: 200px;
  background-color: #4361ee;
  padding: 10px;
}
这是一个块级元素
这是另一个块级元素
常见块级元素 描述
<div> 通用块级容器
<p> 段落
<h1>-<h6> 标题
<ul>, <ol>, <li> 列表元素
<section>, <article> 语义化区块
2. display: inline（内联元素）

内联元素不会独占一行，宽度和高度由内容决定，无法直接设置。

.display-inline {
  display: inline;
  /* width在inline元素上无效 */
  background-color: #f72585;
  padding: 5px;
}

这是一段文本，其中包含内联元素一和内联元素二，它们不会独占一行。

常见内联元素 描述
<span> 通用内联容器
<a> 超链接
<strong>, <em> 强调文本
<img> 图片
<button> 按钮
3. display: inline-block（内联块元素）

结合了inline和block的特性，可以设置宽高，但不会独占一行。

.display-inline-block {
  display: inline-block;
  width: 150px;
  background-color: #90be6d;
  padding: 10px;
  margin: 5px;
}

CSS定位

position属性用于指定元素在页面上的定位方式。结合top、right、bottom、left属性，可以精确控制元素的位置。

1. position: static（静态定位）

默认值，元素按照正常文档流进行布局。top、right、bottom、left属性无效。

.position-static {
  position: static;
  /* top, right, bottom, left属性在static定位中无效 */
  background-color: #4361ee;
  padding: 10px;
}
这是静态定位的元素

静态定位是所有元素的默认定位方式，元素按照正常的文档流进行排列。

2. position: relative（相对定位）

相对定位，相对于元素在正常文档流中的位置进行偏移。

.position-relative {
  position: relative;
  left: 20px;
  top: 10px;
  background-color: #f72585;
  padding: 10px;
}
这是相对定位的元素（向右偏移20px，向下偏移10px）

相对定位会保留元素在文档流中的位置，其他元素不会受到影响。

3. position: absolute（绝对定位）

绝对定位，相对于最近的已定位祖先元素（position不为static）进行定位。如果没有已定位的祖先元素，则相对于初始包含块（通常是viewport）。

.position-absolute {
  position: absolute;
  top: 20px;
  right: 20px;
  background-color: #90be6d;
  padding: 10px;
}
这是绝对定位的元素
这是包含绝对定位元素的容器（position: relative）

绝对定位的元素会脱离正常文档流，不占据空间，可能会与其他元素重叠。

4. position: fixed（固定定位）

固定定位，相对于浏览器窗口进行定位，即使页面滚动，元素位置也不会改变。

.position-fixed {
  position: fixed;
  bottom: 20px;
  right: 20px;
  background-color: #3a0ca3;
  padding: 15px;
  border-radius: 50%;
}

查看页面右下角的固定定位元素（回到顶部按钮）。固定定位的元素会脱离文档流，并且不会随页面滚动而移动。

5. position: sticky（粘性定位）

粘性定位，基于用户的滚动位置来定位。它的行为就像relative，直到滚动到特定阈值点，然后变成fixed。

.position-sticky {
  position: sticky;
  top: 10px;
  background-color: #4cc9f0;
  padding: 10px;
}

向下滚动查看粘性定位效果：

这是粘性定位的元素

滚动结束

粘性定位在滚动到达阈值之前表现为相对定位，之后表现为固定定位。常用于导航栏和标题固定等场景。

6. z-index（层叠顺序）

z-index属性用于控制定位元素的层叠顺序，值越大，元素显示在越上层。

.element-1 {
  position: absolute;
  z-index: 1;
}

.element-2 {
  position: absolute;
  z-index: 2; /* 会显示在element-1上面 */
}

注意：z-index只对定位元素（position值不为static的元素）有效。

Flex弹性布局

Flexbox（弹性盒子）是一种一维布局模型，它使得元素在一个方向上（行或列）的排列更加灵活和高效。

基本的Flex容器
.flex-container {
  display: flex;
  background-color: #f8f9fa;
  padding: 10px;
}
